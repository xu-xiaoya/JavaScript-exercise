例2、菜单

取消项目符号 list-style-type: none;
让li彼此水平相连 float: left;
取消下划线 text-decoration:none; 
将链接转为块级元素 display: block;
设定菜单每项的宽度 width: 100px;
设定菜单的边距 padding: 5px;

<!DOCTYPE html>
<html>
<head>
<meta charset = "utf-8">
<title>菜单</title>
<style>
.nav {
  list-style-type:none; /* 取消项目符号 */
}
.nav li {
  float: left; /* 让li彼此水平相连 */
}
.nav a {
  border: white 1px solid;
  background: purple;
  display: block;   
  width: 100px;
  padding: 5px;
  text-align: center;
  text-decoration: none;  /* 取消下划线 */
  color: white; 
  font-weight: bold;
}
.nav a:hover {
  color: red; 
}
</style>
</head>
<body>
<ul class="nav">
  <li><a href="#">网站首页</a></li>
  <li><a href="#">校园新闻</a></li>
  <li><a href="#">学校概况</a></li>
  <li><a href="#">专业介绍</a></li>
  <li><a href="#">学生活动</a></li>
</ul>
</body>
</html>

例7、表单示例

  margin-left: 150px;
}

  font-size: 14pt;
  color: #36c; 
  text-align: center;
  font-weight: bold;
}

  display: block;
  float: left;
  width: 120px;
  text-align: left;
  margin-left: 20px;
}

  font-weight: bold;
}

</style>  
</head>

  form的特性action：提交后对数据的处理、method：发送服务器的方式、name（id）
  post：将数据隐藏在HTTP头中
-->
<form action="form_action.asp" method="post" name="form1">
<div id="form">
<div id="caption">用 户 注 册</div>

    for：特性的值应该与对应表单控件的name/id特性值相同-->
  <label class="required field" for="user">用户名：</label>
  <!--单行文本输入控件.特性值为“text”，
    name：必要特性，提供向服务器发送的“名称/值”对中的名称部分，值为用户输入-->
  <input type="text" name="user" autofocus required>
  <!--autofocus:当页面打开时自动对焦该控件 
    required：提交时，如果元素中内容为空白不允许提交，同时在浏览器中会弹出提示文字。
  -->
</div>
